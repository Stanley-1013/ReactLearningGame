# React 學習遊戲\n\n**一個模組化的 React 教學闖關遊戲 🎮**\n\n## 📖 專案描述\n\n這是一個完整的 Vite + React 專案，設計用於製作「React 教學遊戲」的前端應用程式。透過模組化闖關的方式，讓學習者循序漸進地掌握 React 的核心概念。\n\n### ✨ 主要特色\n\n- 🎯 **闖關式學習** - 10 個循序漸進的 React 教學關卡\n- 📱 **PWA 支援** - 可安裝到手機桌面，支援離線使用\n- 🔄 **Capacitor 相容** - 可打包成原生手機 App\n- 🎨 **模組化設計** - 清楚的檔案結構，易於擴充\n- 📊 **進度追蹤** - LocalStorage 保存學習進度\n- 🏆 **成就系統** - 學習成就和里程碑獎勵\n- 🎵 **多媒體支援** - 預留圖片、音效、影片插槽\n- 📱 **RWD 響應式** - 完美支援各種裝置尺寸\n\n## 🚀 快速開始\n\n### 系統需求\n\n- Node.js 16.0 或更高版本\n- npm 或 yarn 套件管理器\n\n### 安裝和運行\n\n```bash\n# 1. 複製專案\ngit clone <repository-url>\ncd react-learning-game\n\n# 2. 安裝依賴\nnpm install\n\n# 3. 啟動開發伺服器\nnpm run dev\n\n# 4. 開啟瀏覽器訪問\n# http://localhost:3000\n```\n\n### 建置和部署\n\n```bash\n# 建置生產版本\nnpm run build\n\n# 預覽建置結果\nnpm run preview\n\n# 程式碼檢查\nnpm run lint\n```\n\n## 📁 專案結構\n\n```\nreact-learning-game/\n├── public/                     # 靜態資源\n│   ├── icons/                  # PWA 應用程式圖示\n│   └── manifest.json          # PWA 配置檔案\n├── src/                        # 原始碼目錄\n│   ├── components/             # React 元件\n│   │   ├── HomePage.jsx        # 首頁/關卡選擇\n│   │   ├── ModulePage.jsx      # 單一關卡學習頁面\n│   │   ├── ResultPage.jsx      # 學習進度結果頁面\n│   │   └── ProgressBar.jsx     # 進度條元件\n│   ├── assets/                 # 媒體資源\n│   │   ├── images/             # 圖片資源 (可放 AI 生成圖示)\n│   │   └── audio/              # 音訊資源 (可放 AI 生成音效)\n│   ├── data/                   # 靜態資料\n│   │   └── modules.json        # React 教學關卡資料\n│   ├── App.jsx                 # 主要應用程式元件\n│   ├── main.jsx                # React 進入點\n│   └── index.css               # 全域樣式\n├── index.html                  # HTML 模板\n├── vite.config.js              # Vite 配置 (含 PWA 設定)\n├── package.json                # 專案依賴和腳本\n└── README.md                   # 專案說明文件\n```\n\n## 🎮 功能說明\n\n### 關卡系統\n\n每個關卡包含：\n- **教學內容** - React 概念說明\n- **程式碼範例** - 實際可運行的 React 代碼\n- **互動測驗** - 單選題驗證學習成果\n- **媒體資源** - 圖片、音效支援\n- **進度追蹤** - 完成狀態和解鎖機制\n\n### 學習主題\n\n1. React 是什麼？\n2. JSX 語法介紹\n3. 組件 (Components) 基礎\n4. Props 屬性傳遞\n5. State 狀態管理\n6. 事件處理 (Event Handling)\n7. 條件渲染 (Conditional Rendering)\n8. 列表渲染與 Key\n9. useEffect Hook\n10. 表單處理與受控組件\n\n### PWA 功能\n\n- ✅ **離線支援** - Service Worker 快取所有資源\n- ✅ **應用程式安裝** - 可安裝到裝置桌面\n- ✅ **推播通知** - 學習提醒功能 (可選)\n- ✅ **背景同步** - 離線時保存進度，上線後同步\n\n## 🎨 自訂和擴充\n\n### 新增關卡\n\n編輯 `src/data/modules.json`，按照現有格式新增關卡：\n\n```json\n{\n  \"id\": 11,\n  \"title\": \"新關卡標題\",\n  \"content\": \"教學內容...\",\n  \"codeExample\": \"React 程式碼範例\",\n  \"quiz\": {\n    \"question\": \"測驗問題\",\n    \"options\": [\"選項1\", \"選項2\", \"選項3\"],\n    \"answer\": \"正確答案\"\n  },\n  \"media\": {\n    \"image\": \"/src/assets/images/lesson-11.png\",\n    \"audio\": \"/src/assets/audio/lesson-11.mp3\"\n  }\n}\n```\n\n### AI 資源整合\n\n#### 圖片資源\n- 將 AI 生成的圖示放入 `src/assets/images/`\n- 支援 PNG、JPG、WebP 格式\n- 建議尺寸：關卡圖示 64x64，背景圖 1920x1080\n\n#### 音訊資源\n- 將 AI 生成的音效放入 `src/assets/audio/`\n- 支援 MP3、OGG 格式\n- 建議：背景音樂、按鈕音效、成功提示音\n\n#### 影片資源\n- 元件已預留 `<video>` 標籤\n- 可整合 AI 生成的教學動畫\n- 支援 MP4、WebM 格式\n\n### 主題自訂\n\n在各元件的 CSS 檔案中找到 `TODO` 註解，可自訂：\n- 顏色主題\n- 動畫效果\n- 字體樣式\n- 布局變化\n\n## 📱 手機 App 打包\n\n### 使用 Capacitor\n\n```bash\n# 1. 安裝 Capacitor\nnpm install @capacitor/core @capacitor/cli\nnpm install @capacitor/android @capacitor/ios\n\n# 2. 初始化 Capacitor\nnpx cap init\n\n# 3. 建置網頁版本\nnpm run build\n\n# 4. 新增平台\nnpx cap add android\nnpx cap add ios\n\n# 5. 同步檔案\nnpx cap sync\n\n# 6. 開啟原生專案\nnpx cap open android\nnpx cap open ios\n```\n\n## 🔧 開發工具\n\n### 推薦 VS Code 擴充\n\n- ES7+ React/Redux/React-Native snippets\n- Auto Rename Tag\n- Prettier -
