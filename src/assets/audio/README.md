# Audio Assets 目錄

此目錄用於存放遊戲中使用的音頻資源：

## 用途
- 背景音樂 (BGM)
- 音效 (SFX)
- 語音教學 (Voice)
- 成功/失敗提示音

## AI 生成建議
- 可使用 Suno、AIVA 等工具生成音樂
- 可使用 ElevenLabs 等工具生成語音
- 建議格式：MP3 (相容性佳) 或 OGG (檔案小)
- 音質建議：128kbps (平衡檔案大小與品質)

## 檔案命名規則
- bgm-[theme].mp3 (背景音樂)
- sfx-[action].mp3 (音效)
- voice-module-[id].mp3 (語音教學)

## 注意事項
- 確保音頻已壓縮
- 考慮無音頻模式的使用者體驗
- 提供音量控制選項
